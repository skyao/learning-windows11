## 介绍

https://freefilesync.org/

FreeFileSync是一款文件夹比较和同步软件，用于创建和管理备份 所有重要文件的副本。FreeFileSync不是每次都复制每个文件，而是确定 源文件夹和目标文件夹之间的差异，并仅传输所需的最小数据量。 FreeFileSync是开源软件，适用于Windows、macOS和Linux。
