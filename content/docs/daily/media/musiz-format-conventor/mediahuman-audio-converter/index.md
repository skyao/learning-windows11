## 介绍

https://www.mediahuman.com/audio-converter/

MediaHuman Audio Converter 是一款适用于 macOS 和 Windows 的免费应用程序。有了它，你可以转换 MP3、AAC、WMA、OGG 等格式的音乐，也可以转换 FLAC、Apple Lossless、AIFF、WAV（最高 32 位）等无损格式的音乐。
